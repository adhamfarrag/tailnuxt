### Build Setup

``` bash
# install dependencies
$ npm run install

# serve with hot reload at localhost:3000
$ npm run dev

```

For detailed explanation on how things work, check out [Nuxt.js](https://nuxtjs.org) and [Tailwind](https://nuxtjs.org).
